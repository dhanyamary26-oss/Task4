# 📊 Data Analytics Project 4 – SQL + Python Visualization

##  Project Overview

This project demonstrates how to combine **Python, SQL, and Data Visualization** to analyze sales data and extract meaningful business insights.

The workflow includes:

* Loading and cleaning data from Excel
* Storing data in a SQLite database
* Running SQL queries for analysis
* Visualizing results using Matplotlib
* Interpreting insights from the data

---

##  Technologies Used

* **Python**
* **Pandas**
* **SQLite (sqlite3)**
* **Matplotlib**
* **Excel Dataset**

---

## 📂 Project Workflow

### 🔹 Step 1: Load Dataset

* Imported dataset from Excel using Pandas
* Prepared data for analysis

### 🔹 Step 2: Create SQL Database

* Created a SQLite database (`project4.db`)
* Stored dataset in a table named `orders`

### 🔹 Step 3: SQL Queries

Performed multiple queries to extract insights:

1. Total sales by product
2. Monthly revenue trend
3. Order status distribution by product
4. Quantity vs total price relationship
5. Payment method usage
6. Yearly sales comparison

---

##  Visualizations

The project includes the following charts:

* 📌 **Bar Chart** – Total Sales by Product
* 📌 **Line Chart** – Monthly Revenue Trend
* 📌 **Stacked Bar Chart** – Order Status by Product
* 📌 **Scatter Plot** – Quantity vs Total Price
* 📌 **Pie Chart** – Payment Method Distribution
* 📌 **Bar Charts** – Distorted vs Honest Visualization (Axis Comparison)

###  4. Yearly Sales Comparison

| Year | Sales      |
| ---- | ---------- |
| 2023 | 552,643.24 |
| 2024 | 480,235.87 |
| 2025 | 231,882.85 |

 **Observation:**

* Sales were highest in **2023**
* There is a **declining trend** over the years
* 2025 shows a significant drop, which may indicate:

  * Reduced demand
  * Incomplete data
  * Business performance issues
